\begin{frame}{Problem Statement}
    Why do we need Density Estimation?
    \vspace{0.5em} % Tạo khoảng cách

    \begin{itemize}
        \item \textbf{Data Visualization:} Histograms are sensitive to bin width.
        \item \textbf{Statistical Modeling:} We need a smooth function $\hat{f}(x)$ to estimate probability.
        \item \textbf{Applications:}
        \begin{itemize}
            \item Anomaly Detection (Fraud detection).
            \item Clustering (Mean-shift algorithm).
        \end{itemize}
    \end{itemize}
\end{frame}
